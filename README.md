# OpenMail

## Usage

Run with Roundcube:

```bash
./setup.sh --roundcube
```

Run with SnappyMail:

```bash
./setup.sh --snappymail
```

Run with Sogo:

```bash
./setup.sh --sogo
```

Run just the mailserver:
```bash
./setup.sh
```
